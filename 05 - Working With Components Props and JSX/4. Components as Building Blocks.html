Definition (simple): A component is a reusable piece of UI that includes its own data, logic, and appearance.

What React mainly does: It takes components and renders them to the screen (draws them as a UI).

When you look at a React application, nearly everything you see on the page belongs to a component (or is inside one). That is why components are considered the building blocks of React user interfaces.
