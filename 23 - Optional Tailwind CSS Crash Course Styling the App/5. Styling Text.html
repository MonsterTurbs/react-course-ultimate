1) üß© Tailwind classes in HTML vs JSX

‚úÖ JSX (React)

  • Use className (React prop name).
  • Example: <h1 className="text-xl font-semibold">...</h1>

‚úÖ Plain HTML

  • Use class (standard HTML attribute).
  • Example: <body class="bg-stone-100 text-stone-700">
üß† Why set styles on <body>?
Text color commonly inherits. If the body has text-stone-700, most text automatically uses that color unless overridden. This reduces repeated classes.
<!-- HTML example (global defaults) -->
<body class="bg-stone-100 text-stone-700">
  ...
</body>

// JSX example
export default function Home() {
  return <h1 className="text-xl font-semibold">Hello</h1>;
}

2) ‚úçÔ∏è Common Tailwind text utilities

2.1 üìç Text alignment

  • Center text: text-center
  • Right align: text-right
  • Left align: default (or use text-left)

2.2 üî† Font size

In Tailwind, font size uses the text-* prefix (even though the CSS property is font-size).

  • Small: text-sm
  • Base: text-base
  • Large: text-lg
  • Extra large: text-xl, text-2xl ‚Ä¶ text-9xl
‚úÖ Nice detail
Tailwind sizes often come with sensible default line-heights, which can make text look better immediately without extra work.

2.3 üèãÔ∏è Font weight

Instead of remembering numeric font weights (400/600/700), Tailwind provides names.

  • Regular: font-normal
  • Medium: font-medium
  • Semi-bold: font-semibold
  • Bold: font-bold

2.4 üî° Text transform (uppercase)

  • Uppercase all letters: uppercase
  • Also useful: lowercase, capitalize, normal-case

2.5 ‚ÜîÔ∏è Letter spacing (tracking)

In Tailwind, letter spacing uses tracking-* (CSS property: letter-spacing).

  • Widest example used in lecture: tracking-widest

3) üßØ Arbitrary values (square brackets)

Sometimes you need a value Tailwind doesn‚Äôt provide. Tailwind lets you ‚Äúescape‚Äù the preset scale using square brackets. This is powerful, but should be used sparingly.

‚úÖ Rule of thumb
Use Tailwind presets for ~99% of cases. Use arbitrary values only when you truly need an exact custom number.

Examples

  • Custom letter spacing: tracking-[5px] or tracking-[2rem]
  • Custom font size: text-[100px] (or any value you need)
// Letter spacing ‚Äúescape hatch‚Äù
<a className="tracking-[5px] uppercase">Fast React Pizza</a>

// Custom font size (rarely needed)
<h1 className="text-[100px]">Big Title</h1>
