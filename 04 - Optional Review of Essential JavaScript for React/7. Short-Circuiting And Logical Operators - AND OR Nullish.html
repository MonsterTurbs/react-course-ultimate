üß† What is short-circuiting?

In JavaScript, some logical operators can stop early and return a value immediately. This is called short-circuiting.

Key idea: These operators return a value (not always true/false), and sometimes they don‚Äôt even evaluate the second operand.

üßØ Defaults: || vs ?? (Nullish coalescing)

The nullish coalescing operator ?? is like ||, but safer for numbers and empty strings.

// countWrong becomes "no data" (WRONG, because 0 is valid data)
