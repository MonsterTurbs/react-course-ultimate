ğŸ§  Root layout vs nested layout (mental model)

Root layout lives at app/layout.js. It wraps everything (all top-level routes like /, /cabins, /about, /account).

Nested layout lives inside a specific route segment folder (e.g., app/account/layout.js). It wraps only that segment and its children routes.
