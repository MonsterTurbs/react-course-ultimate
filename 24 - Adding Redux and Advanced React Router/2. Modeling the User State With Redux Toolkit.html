ğŸ§ª Mini mental model (how everything flows)

1) You define the shape of state: initialState (e.g., { username: '' }).

2) You define how state changes: reducers (e.g., updateName).

3) Redux Toolkit generates action creators: userSlice.actions.

4) Store combines reducers and holds the state tree.

5) Provider exposes store to React.

6) Components read with useSelector and write with useDispatch.
