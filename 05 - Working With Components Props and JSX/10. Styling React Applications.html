üßæ Inline styles in JSX (object + double braces)

In plain HTML, inline styles are written as a string: style="color: red;"

Many CSS property names aren‚Äôt valid JavaScript identifiers (example: font-size). React converts these to camelCase for inline styles.

‚ö†Ô∏è JSX rule: className, not class

A common beginner mistake is writing class in JSX like in HTML. React will warn you: Invalid DOM property class. Did you mean className?
