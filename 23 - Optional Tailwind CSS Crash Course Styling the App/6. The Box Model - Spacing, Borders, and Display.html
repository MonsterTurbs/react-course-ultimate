üß± Quick refresher: the CSS Box Model

The box model is how the browser measures and draws elements. In simple terms:

  • Content ‚Üí your text/images
  • Padding ‚Üí space inside the element (between content and border)
  • Border ‚Üí the edge/outline
  • Margin ‚Üí space outside the element (between this element and others)

Tailwind makes the box model fast. Instead of writing CSS like margin-bottom: 32px;, you often write mb-8.

üìè Margin in Tailwind

Pattern: m + direction + -value

m all sides
mx left + right
my top + bottom
mt top
mr right
mb bottom
ml left

‚úÖ Example from the lecture: add spacing below the homepage title

// Tailwind: mb-8 = margin-bottom based on Tailwind's spacing scale

Important: the number is NOT pixels. ‚ö†Ô∏è

Example: mb-8 is not ‚Äú8px‚Äù. It maps to Tailwind‚Äôs spacing scale (often 2rem = 32px in default config).

Common ‚Äúdirection combos‚Äù you will use constantly

  • mb-* ‚Üí push the next element downward
  • mt-* ‚Üí push this element downward from whatever is above it
  • my-* ‚Üí give breathing space above and below
  • mx-auto ‚Üí center a fixed-width element horizontally

üß© ‚ÄúSpace between‚Äù children: space-x / space-y

Tailwind provides a very convenient way to add spacing between direct children without manually adding margins to each child.

üß∑ Borders in Tailwind
