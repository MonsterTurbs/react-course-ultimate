Steps:

1) Create folder: app/cabins
2) Create file: app/cabins/page.js
3) Export a React component from page.js
app/cabins/page.js (example)

Why this works:

Next.js looks for page.js files inside app/. If it finds one, it creates a route. The exported component is rendered for that route.

Server-side note:

In the App Router, these pages are typically React Server Components by default. Meaning: the code can be rendered on the server (more details later in the course).
