1) What is PropTypes?

PropTypes is a way to add runtime type checking for a component‚Äôs props. It lets you say: ‚ÄúThis prop should be a number‚Äù, ‚ÄúThis should be a string‚Äù, etc.

üß† Why this matters

Props are the public API of your component. If a consumer passes the wrong type (e.g., a string instead of a number), PropTypes shows a warning in the console so you can catch bugs earlier.

2) PropTypes vs TypeScript

  • PropTypes ‚Üí Quick runtime warnings, easy to add, but limited and not commonly used across entire apps today.
  • TypeScript ‚Üí Compile-time type safety (stronger), widely used in modern React teams.

‚ö†Ô∏è Important expectation

PropTypes does not prevent your app from running. It primarily provides development warnings. If you want stronger guarantees, TypeScript is the standard approach.

3) Setup: Importing PropTypes

PropTypes comes from a separate package named prop-types. In Create React App, it is often available already, but you still need to import it.

‚úÖ Naming detail

We import PropTypes with a capital P. But the component property is named propTypes (lowercase p).

4) Basic Pattern: Adding propTypes to a Component

You attach an object to your component‚Äôs propTypes property. Each key matches the prop name, and the value is a validator.

  maxRating: PropTypes.number,

‚úÖ Example: Catching a wrong type

If a consumer passes "5" instead of 5, React will warn: ‚ÄúInvalid prop maxRating of type string supplied‚Ä¶‚Äù

5) ‚ÄúisRequired‚Äù and Default Props

You can make a prop required using .isRequired. But if your component already has default values, making props required usually doesn‚Äôt make sense.

  maxRating: PropTypes.number.isRequired,

üß† Practical guidance

Prefer good default values for reusable components. Use .isRequired only when the component truly cannot work without that prop.

7) Common PropTypes Validators (Cheat Sheet)

üßæ Most-used validators

  • PropTypes.string ‚Äî text (e.g., color, className)
  • PropTypes.number ‚Äî numeric values (e.g., maxRating, size)
  • PropTypes.bool ‚Äî true/false flags
  • PropTypes.func ‚Äî function props (callbacks like onSetRating)
  • PropTypes.array ‚Äî arrays (e.g., messages)
  • PropTypes.object ‚Äî objects

üîç Useful ‚Äúadvanced‚Äù ones (good to know)

  • PropTypes.oneOf(["small","large"]) ‚Äî only certain values allowed
  • PropTypes.arrayOf(PropTypes.string) ‚Äî array with item types
  • PropTypes.shape({ id: PropTypes.number }) ‚Äî object with a specific structure
  • PropTypes.node ‚Äî anything renderable (text, elements, fragments)
  • PropTypes.element ‚Äî a React element (e.g., <MyComp />)
