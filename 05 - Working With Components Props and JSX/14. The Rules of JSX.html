üß† Big picture: JSX vs HTML

JSX looks like HTML and is written with a very similar syntax. But JSX is actually JavaScript (an extension of JavaScript syntax).

Why beginners get confused ü§Ø

Because it looks like markup (HTML) but behaves like JavaScript rules.

üîé Common JSX vs HTML differences (reference list)

  • Most DOM attributes use camelCase (e.g., tabIndex)

Styles üé®

  • CSS properties use camelCase: fontSize, backgroundColor

JavaScript in JSX üß†

  • Use { } for expressions/variables
