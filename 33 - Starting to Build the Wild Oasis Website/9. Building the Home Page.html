3) The ‚Äúhero background image‚Äù problem

You want a big background-like image that is responsive and fills a container. With a normal <Image />, Next.js typically expects width and height. But for a hero image, you usually don‚Äôt want fixed dimensions.

‚úÖ Solution: static import + fill

If you statically import the image (instead of using a string path), you can use fill to make it fill its parent container.
